# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 0.1.x   | Yes                |

## Reporting a Vulnerability

If you discover a security vulnerability in ToolGate.NET, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, please use [GitHub Security Advisories](https://github.com/relusion/toolgate-dotnet/security/advisories/new) to report vulnerabilities privately.

### What to include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

### Response timeline

- **Acknowledgment**: Within 48 hours
- **Initial assessment**: Within 1 week
- **Fix and disclosure**: Coordinated with reporter

## Scope

This policy applies to the ToolGate.NET library packages:

- `ToolGate.Core`
- `ToolGate.Policies`
- `ToolGate.Approvals`
- `ToolGate.Adapters.ExtensionsAI`

Sample applications are provided for demonstration purposes and are not covered by this policy.
